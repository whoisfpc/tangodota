# Tango Dota  
## a web base tool for dota2 replay parser

Requirements
---

* node 5+
* npm
* maven
* java 8

Build & Run
---

### 1. Install dependencies
```
npm install
```

### 2. Build java base replay parser
```
cd TangoParser
mvn clean package
```

### 3. Start server
```
npm start
```
